Esta aplicación fue desarrollada en Flutter y funciona como una calculadora de propinas moderna e interactiva. Su objetivo es facilitar el cálculo del monto total a pagar en una cuenta, incluyendo la propina, además de permitir dividir el gasto entre varias personas de manera rápida y visual. El usuario puede ingresar el monto de la cuenta, seleccionar un porcentaje de propina predefinido o introducir uno personalizado, mientras la aplicación actualiza automáticamente los cálculos en tiempo real.
